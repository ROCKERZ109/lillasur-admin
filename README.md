# Lilla Sur Admin

Admin dashboard for managing orders and products for Lilla Sur bakery.

**This is a separate project from the main website** - they share the same Firebase database.

## Features

### Orders Management
- View all orders with customer details
- Filter by status (pending, confirmed, ready, completed, cancelled)
- Search by customer name, email, or order ID
- Update order status with one click
- View order items, notes, and pickup time

### Products Management
- Add, edit, and delete products
- Upload product images to Firebase Storage
- Toggle product availability (show/hide on website)
- Toggle featured status (show on homepage)
- Filter by category
- Search by product name
